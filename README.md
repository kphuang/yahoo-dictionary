# yahoo-dictionary
非官方Chrome擴充功能，不用開啟[Yahoo奇摩字典搜尋](https://tw.dictionary.search.yahoo.com/?_guc_consent_skip=1608245268)就可以速查字詞

## 功能
工具列擴充功能彈出式視窗搜尋中英文字詞

## 如何安裝
1. 開啟Chrome選單
2. 選擇"更多工具"
3. 選擇"擴充功能"
4. 開啟"開發人員模式"
5. 選擇"載入未封裝項目"
6. 選取yahoo-dictionary資料夾
7. 點擊Chrome工具列上的擴充功能圖示
8. 將yahoo-dictionary固定擴充功能
